# 影刃纪元 Era of Shadowblade

## 项目简介

影刃纪元是一个基于 Unreal Engine 制作的第三人称动作游戏练习项目。本项目参考 YouTube 教程完成，主要用于学习虚幻引擎中的角色控制、战斗交互、动画切换、关卡搭建和基础游戏流程制作。

项目以动作冒险玩法为核心，围绕角色移动、攻击、敌人交互、场景探索等内容进行实现，适合作为 Unreal Engine 学习过程中的阶段性作品展示。

## 项目来源

本项目为学习练习项目，制作过程参考了以下 YouTube 教程：

[YouTube 教程播放列表](https://www.youtube.com/watch?v=FNTyIWkv5k8&list=PLiSlOaRBfgkcPAhYpGps16PT_9f28amXi)

项目中的实现内容基于教程学习并进行整理，主要用于课程作业、个人练习和作品集展示。

## 项目内容

- 第三人称角色控制
- 角色移动、跳跃与视角控制
- 基础攻击与动作表现
- 动画蓝图与角色动画切换
- 敌人或目标对象交互
- 游戏场景与关卡搭建
- 基础 UI 或提示信息展示
- 游戏运行流程与关卡体验

## 技术实现

- Unreal Engine：项目搭建、场景制作和游戏运行
- Blueprint：角色控制、交互逻辑和游戏流程
- Animation Blueprint：角色动画状态切换
- Character Controller：第三人称移动与镜头控制
- UMG：界面元素与基础提示展示
- Niagara 或粒子系统：用于特效表现，如攻击、命中或环境效果

## 运行方式

1. 下载或克隆本仓库。
2. 使用 Unreal Engine 打开项目中的 `.uproject` 文件。
3. 等待资源加载完成。
4. 打开主关卡或默认地图。
5. 点击运行，即可体验项目内容。

建议使用与项目开发时一致或更高版本的 Unreal Engine 打开，避免因版本差异导致蓝图、材质或资源加载异常。

## 操作说明

| 操作 | 功能 |
| --- | --- |
| W / A / S / D | 控制角色移动 |
| 鼠标移动 | 控制视角方向 |
| 空格键 | 跳跃 |
| 鼠标左键 | 攻击或触发主要动作 |
| Shift | 冲刺或加速移动 |
| Esc | 暂停或退出相关界面 |

具体按键可能会根据项目实际设置有所不同。
