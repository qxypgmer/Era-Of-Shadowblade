# Era of Shadowblade

## Overview

Era of Shadowblade is a third-person action game practice project built with Unreal Engine. The project was created by following a YouTube tutorial series and is mainly used to practice character control, combat interaction, animation switching, level building, and basic gameplay flow in Unreal Engine.

The project focuses on action-adventure gameplay, including player movement, attacks, scene exploration, and basic interaction logic. It is suitable as a learning project or portfolio piece for Unreal Engine development.

## Tutorial Reference

This is a learning project based on the following YouTube tutorial series:

[YouTube Tutorial Playlist](https://www.youtube.com/watch?v=FNTyIWkv5k8&list=PLiSlOaRBfgkcPAhYpGps16PT_9f28amXi)

The implementation was completed by following and organizing the tutorial content. This repository is intended for coursework, personal practice, and project showcase purposes.

## Features

- Third-person character controller
- Character movement, jumping, and camera control
- Basic attack actions and combat behavior
- Animation Blueprint and animation state transitions
- Enemy or target object interaction
- Game scene and level construction
- Basic UI or on-screen prompts
- Playable gameplay flow and level experience

## Technical Implementation

- Unreal Engine: project setup, scene creation, and gameplay runtime
- Blueprint: character control, interaction logic, and gameplay flow
- Animation Blueprint: character animation state management
- Character Controller: third-person movement and camera control
- UMG: UI elements and basic on-screen prompts
- Niagara or particle systems: visual effects such as attacks, hits, or environmental effects

## How to Run

1. Download or clone this repository.
2. Open the `.uproject` file with Unreal Engine.
3. Wait for all assets to load.
4. Open the main level or default map.
5. Press Play to experience the project.

It is recommended to use the same Unreal Engine version used during development, or a newer compatible version, to avoid Blueprint, material, or asset loading issues.
